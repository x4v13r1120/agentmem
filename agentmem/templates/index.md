# agentmem Index

Quick navigation for this memory system.

- **Working state:** `working/CURRENT.md` — what the agent is doing right now
- **Semantic memory:** `semantic/` — long-term knowledge (facts, patterns, people, mistakes)
- **Episodic memory:** `episodic/` — daily logs of what happened
- **Pending changes:** `pending/CHANGES.md` — changes waiting for review
